# Coffee Sales Analysis Dashboard (Power BI)

## Project Overview

This project features an interactive **Power BI dashboard** built to analyze coffee shop sales data and uncover meaningful business insights. The dashboard uses intuitive visualizations such as bar charts, line charts, pie charts, and slicers to help stakeholders quickly understand sales performance and customer behavior.

The goal of this project is to support **data-driven decision making** by identifying sales trends, popular products, peak hours, and payment preferences.

---

## Key Features

* **Total Sales KPI** to track overall revenue
* **Best-Selling Coffee Analysis** using bar charts
* **Hourly Sales Trends** visualized with line graphs
* **Time Slot Comparison** (Morning, Afternoon, Evening)
* **Payment Method Distribution** (Cash vs Card) using pie chart
* **Interactive Slicers** for:

  * Date range
  * Coffee type
  * Payment method

---

## Insights Generated

* Identified **peak business hours** based on hourly sales trends
* Highlighted **top-performing coffee products**
* Analyzed **customer payment preferences**, showing dominance of card payments
* Compared sales performance across different **time slots of the day**
